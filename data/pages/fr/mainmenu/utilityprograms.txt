====== Outils ======

{{:ic_menu_shopping.svg?nolink&50 |}}

Cette fonction ouvre un écran affichant des applications qui peuvent soit fonctionner en conjonction avec c:geo soit être utilisées pour améliorer certaines fonctionnalités de c:geo. Vous pouvez cliquer sur chaque outil pour ouvrir une page Web avec plus d'informations et/ou possibilités de téléchargement.

^ Logo ^ Nom ^ Description ^
|{{ :send2cgeo.png?nolink&100 |}}|[[https://send2.cgeo.org|Send to c:geo]]|Envoyer à c:geo (//Send to c:geo//) est une extension pour le navigateur **de votre ordinateur**. Lorsque vous naviguez sur geocaching.com, vous pouvez envoyer des caches vers votre appareil Android équipé de c:geo en un simple clic sur un bouton apparaissant dans la page de description de la cache. Les caches ainsi envoyées seront téléchargées par c:geo puis disponibles localement comme si vous les aviez enregistrées directement dans c:geo.|
|{{ :ic_launcher_contacts.png?nolink&100 |}}|[[..:contacts|c:geo contacts]]|Permet d'ouvrir un contact (dans votre carnet d'adresses) directement depuis le nom d'un utilisateur dans une [[..cachedetails#logbook_tab|visite]], de manière à pouvoir plus facilement appeler des amis à l'aide.|
|{{ :helper_brouter.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=btools.routingapp|Navigation BRouter hors ligne]]|BRouter permet la navigation hors ligne. Lorsque BRouter est installé et les fichiers nécessaires téléchargés, c:geo vous indiquera sur la [[livemap|carte]] l'itinéraire vers la cache.\\ Un petit guide expliquant comment BRouter fonctionne avec c:geo peut être trouvé [[..:brouter|ici]].|
|{{ :helper_pocketquery.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=org.pquery|Pocket Query Creator]]|Permet la création facile et le téléchargement de Pocket Queries centrées sur votre position actuelle ou un point sélectionné sur la carte. Nécessite un compte Premium Geocaching.com.|
|{{ :helper_google_translate.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=com.google.android.apps.translate|Google Translator]]|Si vous téléchargez des modules de traduction dans l'application Google Traduction, vous pourrez facilement traduire les descriptions des caches dans c:geo par un appui long sur le texte de la description (même sans connexion Internet).|
|{{ :helper_where_you_go.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=menion.android.whereyougo|WhereYouGo]]|WhereYouGo permet de rechercher et de jouer avec les caches de type Wherigo. c:geo peut automatiquement lancer le [[..navigation|téléchargement de cartouches]] utilisables ensuite dans l'application WhereYouGo.|
|{{ :helper_gpsstatus.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=com.eclipsim.gpsstatus2|GPS Status & Toolbox]]|Vous pouvez utiliser le radar de cette application dans c:geo. Il gère des informations supplémentaires relatives au GPS.|
|{{ :helper_bluetoothgps.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=googoo.android.btgps|Bluetooth GPS]]|Vous permet d'utiliser un GPS externe pour une localisation plus précise tout en économisant la batterie de votre appareil Android.\\ \\ :!: Veuillez noter que vous devez désactiver l'utilisation des services Google Play dans les [[settings#various_other_system_settings|paramètres]] de c:geo pour utiliser un GPS Bluetooth.|
|{{:helper_gps_locker.png?nolink&100|}}|[[https://play.google.com/store/apps/details?id=com.silentlexx.gpslock|GPS Locker]]|Le verrouillage GPS conserve le signal GPS actif lors du basculement entre les applications ou si l'écran de votre appareil s'éteint.|
|{{ :helper_barcode.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=com.google.zxing.client.android|Barcode Scanner]]|Il existe des programmes et des sites web permettant de générer un QR code contenant un géocode. En installant cette application, vous pourrez lire des QR codes et ouvrir les caches correspondantes dans c:geo.|
|{{ :helper_chirp_wolf.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=de.wolffire.chirpwolf|Chirp Wolf]]|Chirp Wolf vous permet de lire les données des balises ANT+/Garmin chirp, ce qui est nécessaire pour les géocaches ANT+/chirp.|
|{{ :helper_locus.png?nolink&100 |}}|[[https://play.google.com/store/apps/details?id=menion.android.locus|Locus]]|Application pour la navigation en plein air pour téléphone ou tablette. Visualisez les cartes topo hors-ligne, enregistrez le chemin parcouru, cherchez les géocaches, utilisez le guidage vocal et plus encore.|
